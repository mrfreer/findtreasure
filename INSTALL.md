# Find Treasure - Installation Guide

## System Requirements
- Windows 10 or Windows 11
- DirectX 11 compatible graphics card
- 2GB RAM
- 100MB free disk space

## Installation Instructions

### Option 1: Using the Installer (Recommended)
1. Download the `FindTreasureSetup.exe` installer from the releases page
2. Run the installer and follow the on-screen instructions
3. The game will be installed to your Program Files folder by default
4. A shortcut will be created in your Start Menu and optionally on your Desktop

### Option 2: Manual Installation
1. Download the ZIP file containing the game files
2. Extract the ZIP file to a location of your choice
3. Run `SideScroller.exe` to start the game

## Playing the Game
- Use the arrow keys to move the player character
- Jump with the spacebar
- Reach the flag at the top of the randomly generated stairs to win
- If you fall off the screen, you'll respawn in the middle

## Troubleshooting
- If the game doesn't start, make sure you have the latest DirectX installed
- If you encounter any issues, please report them on the GitHub repository

Enjoy the game!
